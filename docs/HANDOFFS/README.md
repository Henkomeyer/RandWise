# Agent Handoffs

This directory contains per-task handoff notes for RandWise agents.

Each handoff should be concise and use the required structure from [AGENT_PLAN.md](../AGENT_PLAN.md):

```markdown
## Completed
## Files changed
## API or schema decisions
## Tests run and results
## Known limitations
## Risks
## Required follow-up
```

Handoffs should identify changed files, decisions made, validation performed, known risks, and dependencies for later agents. They should not replace canonical source-of-truth documents such as API, schema, UX, task graph, or implementation status documents.
