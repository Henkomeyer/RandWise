## API or schema decisions
`GET /api/v1/profile/export` returns a JSON account bundle excluding encrypted phone numbers, raw WhatsApp text and token values. `DELETE /api/v1/profile` removes owned financial data, revokes stored refresh tokens, anonymises the retained app-user shell and blocks future login with the original email.
